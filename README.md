# Travel Recommendation System (DSA Mini Project)

This is a C++ + Qt based Travel Recommendation System created as a Data Structures mini project.

## Features
- Linked List for storing places
- Insertion Sort (sort by budget)
- Binary Search (search by budget)
- Stack (recently searched places)
- Qt GUI frontend

## Technologies Used
- C++
- Qt Widgets
- MinGW

## How to Run (Easy Way)

1. Go to the `release` folder
2. Double click `travelGUI.exe`

(No Qt installation required)

## How to Run From Source

1. Open Qt Creator
2. Open the `.pro` file
3. Click Run

## Dataset

Includes multiple tourist destinations with:
- Name
- Budget
- Rating

## Data Structures Used

- Linked List
- Stack
- Array
- Binary Search
- Insertion Sort
